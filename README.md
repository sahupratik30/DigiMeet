   - ### Key Features
      ```
      1. Real-time audio and video streaming.
      2. Real-time chat.
      3. Supports multiple participants.
      4. Easy to use.
      ```
                                                         
## Technology Stack      
   - ### Front-End
      ```
      1. HTML
      2. CSS
      3. Vanilla JavaScript
      4. EJS(templating engine for server side rendering)
      ```
   - ### Back-End
      ```
      1. NodeJS
      2. ExpressJS
      3. WebRTC --> PeerJS(for real-time peer to peer communication)
      4. Socket.io
      ```
